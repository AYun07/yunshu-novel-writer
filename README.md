## ✨ 功能特性

### 🤖 AI智能创作
- **多模型支持** - 支持 GPT-4、Claude、DeepSeek 等主流AI模型
- **智能大纲生成** - AI自动生成小说大纲
- **章节内容创作** - 流式生成，实时预览
- **角色设定生成** - 快速创建丰富角色
- **世界观构建** - 构建完整小说世界

### ✍️ 创作工具
- **专业编辑器** - 富文本编辑，支持Markdown
- **提示词库** - 精心整理的创作模板
- **语料库管理** - 个性化训练素材
- **写作目标** - 进度跟踪，培养创作习惯

### 🎨 界面特色
- **多主题切换** - 5种主题风格自由切换
- **响应式设计** - 完美适配各种设备
- **暗黑模式** - 护眼夜间写作

## 🚀 快速开始

### 环境要求
- Node.js 18+
- pnpm (推荐) 或 npm

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/AYun07/yunshu-novel-writer.git

# 进入项目目录
cd yunshu-novel-writer

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev
```

### 配置API

1. 启动项目后，点击右上角「配置API」
2. 输入您的API地址和密钥
3. 选择默认模型
4. 开始创作！
